# Security Policy (Educational / Defensive Repository)

## Scope
This repository contains **defensive OSINT / CTI analysis** only.
- Passive OSINT only
- Public sources only
- No scanning, exploitation, credential testing, or intrusion

## Responsible Use
Do not use this repository to:
- Target systems, individuals, or accounts
- Conduct unauthorized testing
- Attempt access to restricted resources

## Reporting Concerns
If you believe any content here is unsafe or violates ethical boundaries:
- Open a GitHub issue labeled **security**
- Or contact the maintainer directly (via GitHub profile)

## Data Handling
- Evidence is limited to **screenshots/outputs from public sources**
- No credentials, private keys, or restricted data is included intentionally
